Manufacturer

The goal of a manufacturer is to specify the design of a single hardware or software component. An example of such a component is a chip or solar inverter. The manufacturer specifies the input and output ports and the default values of any constants that can be wired to these ports.

Start as Manufacturer »

Engineer

The goal of an engineer is to group together a set of components designed by one or more manufacturers and to create a packaged solution. An example of such a solution is a micro-controller that controls multiple electric vehicle charging stations. The engineer wires together a group of components and can instantiate constants if required.

Start as Engineer »

Technician

The goal of a technician is to install a prepackaged solution (designed by an engineer) in the field. The technician is only interested in answering a predefined set of questions that will fill in missing constant values, instantiate components dynamically or wire components together.
